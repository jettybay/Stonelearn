# Landing components (folderized)

This folder contains small, standalone landing-page UI pieces under:

- `navbar/`
- `section-header/`
- `feature-card/`
- `dashboard-preview/`
- `branding-switcher/`
- `pricing-card/`
- `faq-item/`
- `animated-counter/`
- `footer/`

These components are currently additive and do not replace the existing `src/components/landing/*.tsx` landing composition.
